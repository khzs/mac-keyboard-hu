# mac-keyboard-hu

Hungarian PC keyboard layout for OS X and macOS

## Installation

```console
cp Hungarian-Win.keylayout ~/Library/Keyboard\ Layouts
cp Hungarian-Win.icns ~/Library/Keyboard\ Layouts
```
